# django-simple-icons

Use [Simple Icons](https://simpleicons.org/) — 3,400+ SVG brand icons — in your Django
and Jinja templates. Icons are bundled in the package, rendered inline as SVG, and
follow the surrounding text color by default.

Full usage documentation arrives with the first release.
